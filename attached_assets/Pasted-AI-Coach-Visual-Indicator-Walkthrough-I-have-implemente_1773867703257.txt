AI Coach Visual Indicator - Walkthrough
I have implemented the visual indicator to let users know when the AI Coach is running in the background. Note that a database migration will be required.

What was changed
1. Database Schema
Added isAutoCoaching to the users table in 
shared/schema.ts
 to persistently track when the AI coach is running.
Action Required: Since I did not have access to the DATABASE_URL environment variable, I could not automatically run the drizzle migration. Please run the following commands to apply the schema change:
bash
npx drizzle-kit generate
npx drizzle-kit push
2. Backend Service (
server/services/coachService.ts
 & 
server/storage/users.ts
)
Implemented 
updateIsAutoCoaching
 in the storage layer.
triggerAutoCoach
 now safely sets the isAutoCoaching flag to true when it starts and resets it to false when it finishes.
3. Frontend Polling (
client/src/hooks/useAuth.ts
)
Added a 
refetchInterval
 to the user auth query that activates (polling every 2s) when isAutoCoaching is true.
Integrated useQueryClient to watch for when isAutoCoaching transitions from true to false. Once this happens, the timeline data (/api/v1/timeline) is automatically invalidated and re-fetched to show the AI's new workout modifications!
4. UI Indicator (
client/src/pages/Timeline.tsx
, 
client/src/components/timeline/TimelineWorkoutCard.tsx
)
The UI indicator is now rendered directly inside the specific upcoming workout cards that the AI Coach is working on (the next 7 planned schedule days).
The entire targeted card is now fully animated: It receives a pulsing, glowing primary border, a glowing shadow, and scales up slightly while the AI Coach is running.
It also displays an animated AI Modifying badge right alongside the Planned badge.
This indicator automatically displays when user.isAutoCoaching is true and disappears when the AI Coach finishes.
How to test this
Apply the database change (npx drizzle-kit generate/push).
Start the development server (npm run dev).
Create/Log a new workout on the timeline.
Immediately after logging the workout, the specific targeted upcoming workouts (up to 7 days ahead) will display the "AI Modifying" animated badge while the coach applies suggestions in the background.
Once the badges disappear, your upcoming workouts will be refreshed and should show the AI Coach's modifications!
